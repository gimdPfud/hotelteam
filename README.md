# 🏨 HotelTeam - 호텔관리와 게시판

학원에서 배운 기술을 활용하여 제작한 호텔 관리 시스템과 게시판이 결합된 프로젝트입니다.

## 소개
* 프로젝트명: HotelTeam (호텔관리와 게시판)
* 개발 형태: 5인 1조 팀 프로젝트
* 개발 기간: 1일
* 목적: 대규모 팀 협업 및 복합 시스템 구현 실습

## 사용 기술
* Language: Java 17
* Framework: Spring Boot
* Database: MariaDB
* Frontend: HTML
* Template Engine: Thymeleaf

## 주요 기능

### 호텔 관리 시스템
* 객실 정보 관리
* 예약 관리
* 체크인/체크아웃 처리
* 고객 정보 관리

### 게시판 기능
* 공지사항 관리
* 문의게시판
* 리뷰 및 후기

### 통합 관리
* 관리자 대시보드
* 사용자별 권한 관리
